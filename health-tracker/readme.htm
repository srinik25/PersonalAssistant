Health Tracker

A locally-hosted personal health tracking site with AI-powered insights.

How to Start

Double-click Start Health Tracker.command or run in Terminal:

cd ~/Documents/PersonalAssistant/health-tracker && python3 -m http.server 8081

Then open http://localhost:8081 in your browser.

How to Stop

Press Ctrl + C in the Terminal window running the server.

Note: The server must be running for the site to work. After a reboot, re-run the command above. The API key is stored in js/secrets.js.

Features

  • Dashboard with health categories overview
  • 11 health categories: Vitals, Vaccines, Cardio, Lipid, Gastro, Physio, Teeth, Eyes, Ears, Sleep, Other
  • Add/delete entries per category with freeform values and notes
  • AI-powered summaries per category and overall analysis
  • Health profile with goals generation
  • PDF download for records and analyses
  • PII stripping before any LLM calls

Pages

  • Dashboard (index.html) - Overview of all categories
  • Category (category.html?cat=vitals) - View/add entries per category
  • Analysis (analysis.html) - Overall health analysis
  • Profile (profile.html) - Health profile and goals
