Finance Tracker

A locally-hosted financial analysis site with AI-powered insights.

How to Start

Open Terminal and run:

cd ~/Documents/PersonalAssistant/finance-tracker && python3 -m http.server 8080

Then open http://localhost:8080 in your browser.

How to Stop

Press Ctrl + C in the Terminal window running the server.

Note: The server must be running for the site to work. After a reboot, re-run the command above. The API key is stored in ~/Documents/PersonalAssistant/secrets.js.

Features

  • Dashboard with portfolio overview and trends
  • Account management (brokerage, IRA, Roth IRA, annuity, CD, crypto, MMA)
  • Per-account holdings with cost basis tracking
  • AI-powered analysis (no PII sent to LLM)
  • Financial profile with goals and risk tolerance
  • Snapshot-based trend tracking over time
